## 一、前言

　　COVID-19 在 2020 年初急速擴散，成為全球前所未見的公共衛生危機，世界各國股市都受到強烈震盪。美股在短時間內多次熔斷，全球金融市場陷入恐慌。然而台灣的股市表現卻相對特殊：雖然初期下跌很深，但很快就反彈，甚至在 2020–2021 期間多次創下新高。這樣的走勢引起我對台灣金融市場韌性的好奇。

　　我會選擇這個題目主要有三個理由：第一，我自己對投資有興趣，也注意到身邊台灣朋友在疫情期間開始開戶、追蹤股市。第二，台灣股市高度依賴科技與半導體，而疫情反而讓科技需求增加，使得台灣的相關產業受惠。第三，我想用更生活化但有結構的方式了解：究竟是什麼讓台灣股市在疫情期間能快速恢復？

<div style="padding:10px 14px;border:1px solid #d0d4ed;border-radius:8px;background:#fafbff;margin-top:10px;margin-bottom:18px;">

1. 查閱書面資料（證交所、主計總處、台灣財經新聞）。<br>

## 二、背景與整體狀況

### （一）疫情全球影響與台灣防疫情況

　　疫情爆發後，歐美國家面臨封城、醫療崩潰、經濟活動停擺，使得股市出現劇烈下跌與反覆震盪。相較之下，台灣早期的防疫表現亮眼，社區感染數相對低、生活受到的影響較小，因此整體經濟活動並未像其他國家一樣完全停止。

　　儘管如此，台灣仍無法避免全球性恐慌帶來的股市影響。在 2020 年 3 月，台股也跟著全球一起大跌，但後續恢復速度明顯較快。

### （二）台灣股市結構特色

1. **半導體與電子產業佔比高**  
   台積電、聯發科等企業在全球供應鏈中占有重要地位，使得科技股成為台股的最大支撐。

2. **出口導向、外需強烈影響**  
   全球科技需求復甦時，台灣的科技股往往能快速受惠。

3. **散戶參與度高、手機 App 下單普及**  
   使得市場活絡，但也可能加劇短期波動。

### （三）疫情前後的台股走勢

| 年份 | 主要事件 | 年初 | 最低點 | 年底 | 年度情況 |
| ---- | -------- | ---- | ------ | ---- | -------- |

## 三、疫情對台灣股市的主要影響（重點分析）

### （一）半導體支撐台股：台灣的核心優勢

　　疫情讓全球對科技設備需求暴增，包括筆電、雲端設備、資料中心、伺服器、視訊設備等，這些產品都需要大量半導體。台積電的先進製程需求在疫情期間不減反增，使其股價快速反彈。

<div style="padding:10px 14px;border:1px solid #d0e6ff;background:#f8fbff;border-radius:8px;margin:14px 0;">
<strong>科技需求上升 → 台積電強勢 → 台股跟著上漲</strong>
</div>

### （二）政策與資金面：市場「錢太多」

　　台灣央行與全球各國一樣，在疫情期間採取寬鬆政策並調降利率，讓存款利息幾乎接近 0%。  
　　旅遊被迫取消、娛樂支出下降，使得部分家庭的「閒置資金」增加。

這些資金最終湧入股市，使得成交量大幅擴張。

### （三）投資人行為變化：散戶湧入、手機 App 佔主流

| 行為項目 | 疫情前 | 疫情期間 | 原因 |
| ------- | ------ | ---------- | ---- |
| 新開戶數 | 穩定成長 | 大幅增加 | 在家時間變多、社群分享多 |
| 手機下單 | 年輕人為主 | 廣泛普及 | App設計更直覺 |
| 當沖交易 | 較少 | 明顯變多 | 波動大、想賺價差 |
| 長期投資 | 穩定 | 有下降 | 熱門題材吸引追高 |

### （四）情緒心理：從恐慌到 FOMO

　　疫情初期市場恐慌，許多人在低點賣出；然而當台股一路彈回甚至創新高，社群媒體到處都在討論投資，形成「害怕錯過」（FOMO）的心理，使得更多新手在高點追入。

<div style="padding:10px;border-left:4px solid #ff4b4b;background:#ffecec;margin:12px 0;">
<strong>FOMO 心態：</strong>「大家都在賺，我是不是也應該買？」 → 高點進場風險變大
</div>

### （五）訪談：三位 UC Berkeley 台灣經濟系學生的看法

<div style="padding:12px;border:1px solid #d0d4ed;background:#f8f9ff;border-radius:8px;margin-bottom:12px;">
<strong>學生 A（大二、台北）</strong><br>
疫情前完全不懂股市，疫情後因為看到台股跌深反彈才開始研究。上 Berkeley 的總體課後，他發現課本裡的供需、利率、預期等概念，都能用來解釋台股的變化。他以小額 ETF 入門，目標是不被短期波動影響。
</div>

<div style="padding:12px;border:1px solid #d0d4ed;background:#f8f9ff;border-radius:8px;margin-bottom:12px;">
<strong>學生 B（大三、台中）</strong><br>
原本就有關注金融新聞。疫情期間因為線上課變多，他花更多時間研究利率與全球資金。認為台股強勢不只是半導體本身強，而是「國際資金尋找相對安全的市場」。偏向長期持有大型權值股。
</div>

<div style="padding:12px;border:1px solid #d0d4ed;background:#f8f9ff;border-radius:8px;margin-bottom:12px;">
<strong>學生 C（大四、高雄）</strong><br>
喜歡做資料分析。他下載台股指數與成交量資料後發現：2021 年成交量是疫情前的好幾倍。他認為散戶大量湧入可以讓市場更活絡，但也可能「熱過頭」。提醒新手不要只看社群，而要看基本面。
</div>

## 四、總結

　　整體而言，COVID-19 對台灣股市的影響可分為「短期劇烈震盪」與「中長期強勢上漲」。台灣受到全球市場恐慌拖累，但因科技產業需求強勁、資金充沛、散戶大量進場，股市在疫情後期表現亮眼。

　　不過，強勢走勢並不代表所有投資人都賺錢。許多新手在最高點追入、或在低點恐慌賣出。這提醒我們：  
　　**投資需要長期思考、分散風險，不應過度受情緒影響。**

<div style="padding:10px 14px;border-left:4px solid #27ae60;background:#ecf9f0;margin:14px 0;">
<strong>這次研究我學到：</strong><br>
1. 台灣的半導體結構是真正支撐市場的核心。<br>
2. 利率與資金面是理解市場的重要關鍵。<br>
3. 散戶情緒比教科書更有力量，心理面非常重要。<br>
4. 未來面對突發事件，最重要的是策略紀律與心態管理。
</div>
